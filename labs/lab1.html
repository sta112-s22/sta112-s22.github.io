  3. It can also be helpful to calculate summary statistics for distributions. Add the following code to your lab report, and run:

penguins %>%
  pull(body_mass_g) %>%
  mean()

However, we dont get a number when we run this code. What result do you get?

Whats going on here?

mean is a function for calculating the mean, and pull extracts a column from the dataset. So penguins %>% pull(body_mass_g) %>% mean() takes penguins, extracts the body mass column, and then calculates the mean. Notice that pipes can be chained together!

Remember, you can learn more about a topic in R with ?. Run ?NA in the console to learn more about missing values.

  1. NA in R means Not Available and results from missing values in the data. Typically, if data contain NAs, then a summary of that data (e.g., the mean) will also contain NAs. There are a wide variety of methods for handling missing data, but for now we will just remove these rows from the dataset. Add the following code to your lab report, and run
penguins_no_nas <- penguins %>%
  drop_na()

(theres no question to answer on this exercise).

Whats going on here?

drop_na is a function that removes any rows containing NAs from the dataset. So penguins %>% drop_na() means take penguins, THEN remove the missing values. The <- is an assignment operator. It means store the result. So, the code creates a new data table by removing the missing values from penguins, and saves the new data table as penguins_no_nas.

  3. Of course, penguins_no_nas still has penguins from three different species. As we saw on Wednesday, the different species can have pretty different characteristics! So maybe it doesnt make sense to group all three species together into one histogram. Add and run the following code for Exercise 8:

penguins_no_nas %>%
  ggplot(aes(x = body_mass_g)) +
  geom_histogram() +
  facet_wrap(~species)

How does the distribution of body mass differ between species?

Whats going on here?

ggplot builds plots layer by layer; the + means add another layer. Weve already seen geom_histogram used to create histograms. Now were adding a faceting layer, which splits the plot by species. Well learn more about ggplot and layers next week.

  1. If were going to look at the distribution of body mass separately for each species, we should also calculate summary statistics separately for each species. Add and run the following code:
penguins_no_nas %>%
  group_by(species) %>%
  summarize(mean(body_mass_g))

Report the mean for each species.

Whats going on here?

summarize calculates summary statistics for a dataset. In summarize, we specify which statistics to calculate, and on which columns. group_by is often used before summarize, and it means group the data by the values of a column. So, the code above means take penguins_no_nas, group the penguins by their species, and within each group calculate the mean body mass.
