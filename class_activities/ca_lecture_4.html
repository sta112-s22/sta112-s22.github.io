Class Activity, January 19

In this activity, you will practice using ggplot to create and customize visualizations. You will use the penguins data from last week and todays class. Work in groups to answer the questions below. Solutions are provided for the first two questions, in case you get stuck.

Setup

As with all class activities, there is nothing you will need to submit. However, in case it helps you work on the questions, an R Markdown template is provided for you: activity template

The penguins data comes from the palmerpenguins package, and ggplot is included in the tidyverse package. You will need to load these packages (with library) to run the code today. If you use the R Markdown template, those packages are included in the setup chunk, so you just need to run the setup chunk first.
